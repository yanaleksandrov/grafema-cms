Accordion

collapsible content panels

Displays collapsible content panels for presenting information in a limited amount of space. Click headers to expand/collapse content that is broken into logical sections, much like tabs.

Allows you to expand and collapse elements using smooth animations.

Usage

The primary API for using this plugin is the x-collapse directive. x-collapse can only exist on an element that already has an x-show directive. When added to an x-show element, x-collapse will smoothly "collapse" and "expand" the element when it's visibility is toggled by animating its height property.
