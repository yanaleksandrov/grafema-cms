Select

drop-down list

The <select> allows you to create an interface element in the form of a drop-down list, as well as a list with one or multiple selections.

Default selects

Each item is created using the <option> element, which must be nested in the <select> container. If you plan to send the list data to the server, then you need to put <select> inside the form. This is also necessary when the list data is accessed through scripts.



With Slim Select library

Currently, the select tag is poorly stylized and has many functional limitations. Therefore, the Slim Select library is used, which expands the capabilities of the list.

Basic single select usage with Slim Select is extremely easy. Just set the attribute x-data="slimselect" to your select element. Multi selects work just the same as a single select. The only difference is setting the multiple attribute on your select.



Slim Select options

You can use any of the available library options. See the full list on the official website.
