Syntax highlight

lightweight inline messages

Syntax highlighting is based on the prism.js library. In the current configuration supports html, css, js, and json. To get started, wrap the code in the pre tag and add the x-highlight directive.

Usage

The highlighting language is selected by adding a modifier, see the examples below.

<pre x-highlight.js>import {FormControl, FormGroup} from '@angular/forms';

export class LabelExample {
    readonly testForm = new FormGroup({
        testValue: new FormControl(),
    });
}</pre>
