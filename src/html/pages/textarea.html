Textarea

multiline text input

Textarea Input for multiline text, not for rich text entry.

Basic

Simple textarea with fixed height. The textarea is fixed as 2 lines default.

Disabled

To make textarea as disabled, add disabled property to the Textarea.

With autoresize

Automatically adjusted the height of textarea according to the number of lines without limited. Just add the x-textarea directory for the text field.

The maximum number of rows can be set as the attribute value portion of the directive. In example, where 5 is the maximum number of rows: x-textarea="5". The minimum value of rows is set via the rows attribute.
