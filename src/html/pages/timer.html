Timer

time counter before the event

Counting time in four different units: seconds, minutes, hours and days. The timer is useful to use on pages such as "Soon on sale" or "Maintenance". It allows the user to visually determine the time before an event.

Basic Usage

Starting a timer is very simple. Declare a new Alpine component and its data for a block of HTML: x-data="timer(...)" and add x-text attribute.

The timer function takes 2 parameters: endTime and startTime and is set in the ISO 8601 format. The required endTime parameter sets the end time of the countdown. The startTime parameter is optional, and sets the timer start time: immediately (if the parameter is not specified) or delayed.

Attributes usage

Days, hours, minutes and seconds are output via the x-data attribute with the values day, hour, min and sec. At the same time, the nesting is not limited by anything. You can create html elements of any complexity.
